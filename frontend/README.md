# ScanAhead Frontend

React + Vite + TypeScript medical pre-diagnosis UI with Supabase auth, patient capture flow, and doctor dashboard.

## Setup

1. Install dependencies:
   - npm install
2. Add Supabase keys in .env.local:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
3. Start the dev server:
   - npm run dev

## Features (Initial)

- Role-based routing for patients and doctors
- Patient intake: appointment form, body part selection, photo capture, voice note
- Doctor dashboard with case list and detail view
- 3D model viewer placeholder with orbit controls

## Scripts

- npm run dev
- npm run build
- npm run lint
